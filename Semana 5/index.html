Inicio

Bienvenidos a FitZone Store, una página web dedicada a la venta de artículos, accesorios y productos relacionados con el entrenamiento físico, tanto para personas que entrenan en casa como para quienes asisten al gimnasio.

Quiénes Somos

FitZone Store es un proyecto web enfocado en ofrecer productos útiles para el entrenamiento físico y el bienestar personal. Nuestro objetivo es brindar a los usuarios una alternativa práctica para conocer artículos deportivos como bandas elásticas, mancuernas, colchonetas, botellas de agua, guantes de gimnasio y otros accesorios que facilitan la actividad física.

Esta página también busca orientar a las personas que desean iniciar una rutina de ejercicios de manera organizada, ya sea desde casa o en un gimnasio. Como proyecto académico, permite aplicar los conocimientos de HTML5, CSS3, Bootstrap y JavaScript mediante una estructura clara, ordenada, visual, responsiva y dinámica.
